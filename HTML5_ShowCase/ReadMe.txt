This project uses JQuery UI Tabs and displays some modified versions of the finished projects in the HTML5_BuildingBlocks package.  

Again, the materials found in HTML5_BulidingBlocks have been inspired by the New Boston Turorial: HTML5
This totorial can be found here: https://www.youtube.com/playlist?list=PL081AC329706B2953
Please support the series as it's a very usefull tool to learn HTML5 and other web development items.

Please note, that some of these examples use the chrome web-kit and won't display properly on other browsers.

To view the showcase, please open index.html in the chrome browser.
