The Flex Box Model shows off the power of the chrome webkit. Below is a sample webpage and the articles are sized dynamically through the web-kit horizontal orientation property. To see how it works try and resize the window and the acticle boxes will change size with the window size.

Please note that web-kit properties only work in the Chrome Browser.

This example shows the effects of the web-kit shadow property.

  • The first box, a simple text box with a shadow property added to the text.
  • The second box has the same properties as the first but has an added black-white gradient on the box id.
  • The third box has the web-kit properties of rounded edges and a center gradient of blue and green.

Please note that web-kit properties only work in the Chrome Browser.

The web-kit transfroms will allow dynamic changes to items. These examples will cover hover over functionality.

  • Box1 will rotate 45 degrees.
  • Box2 will increase in size.
  • Box3 will reverse.
  • Box4 will skew.
  • Box5 will rotate 360 degress and change color.

Please note that web-kit properties only work in the Chrome Browser.
